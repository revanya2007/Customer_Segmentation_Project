Customer Segmentation Project analyzes customer demographics and purchasing behavior to identify distinct customer groups. Using clustering techniques and data visualization, the project helps understand customer preferences and supports targeted marketing strategies.
# Customer_Segmentation_Project

## Project Overview

This project focuses on customer segmentation using demographic and behavioral data. The objective is to group customers based on characteristics such as age, income, spending score, and purchase frequency to better understand customer behavior and support business decision-making.

## Features

* Customer Demographic Analysis
* Purchase Behavior Analysis
* Customer Segmentation
* Data Visualization
* Business Insights Generation

## Dataset Information

The dataset contains 1,000 customer records with:

* Customer ID
* Age
* Gender
* Region
* Annual Income
* Spending Score
* Purchase Frequency
* Customer Segment

## Tools Used

* Microsoft Excel
* Python (Scikit-learn)
* GitHub

## Expected Outcome

Identify customer groups, analyze purchasing patterns, and generate insights that can help businesses improve marketing strategies and customer engagement.
